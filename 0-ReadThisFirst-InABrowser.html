This folder has practice problems for the Paper-and-Pencil part of Exam 3, along with links to answers.

See your instructor for what parts of these practice problems, if any, are part of your Admission Ticket for Exam 3.

For the Paper-and-Pencil part of the exam, the only external resource that you may use is THREE (one-sided) 8.5 x 11 sheets of paper, with whatever you want on them (printed or handwritten).

The exam will assess your ability to trace by hand code that includes:

  • WHILE loops, in both the while True and while condition forms.
  • Simple loops within loops.
  • A class definition and code that constructs multiple instances of that class, accesses their instance variables, and calls their methods. (This will require that you include a box-and-pointer diagram.)
  • Code that has a function whose arguments include container objects that are mutated within the function. (This will require that you include a box-and-pointer diagram.)

The relevant problems in the attached PDFS are problems 2, 3 and 6 of the attached Exam3_Paper_and_Pencil_SAMPLE_EXAM.pdf and the sole problem in AnotherPracticeProblem.pdf.
